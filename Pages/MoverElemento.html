Mover Elementos en la Pantalla: Funcion Flex

Directions

Direcciones: Esta variable define una lista de posibles valores para la propiedad flex-direction, que determina la dirección en la que los elementos flexibles se colocan dentro del contenedor flex.

flex-row Los elementos se colocan en filas, desde el inicio hasta el final del contenedor.

Elemento 1
Elemento 2
Elemento 3
            <div class="flex flex-row flex-gap-1">
                <div class="badge primary">Elemento 1</div>
                <div class="badge primary">Elemento 2</div>
                <div class="badge primary">Elemento 3</div>
            </div>

flex-column Los elementos se colocan en columnas, desde el inicio hasta el final del contenedor.

Elemento 1
Elemento 2
Elemento 3
            <div class="flex flex-column flex-gap-2">
                <div class="badge secondary">Elemento 1</div>
                <div class="badge secondary">Elemento 2</div>
                <div class="badge secondary">Elemento 3</div>
            </div>

Tambien puedes usar las clases: flex-row-reverse y flex-column-reverse para filas y columnas respectivamente, pero en orden inverso al valor predeterminado.

Justifies

Justifies: Esta variable define una lista de posibles valores para la propiedad justify-content, que alinea los elementos flexibles a lo largo del eje principal del contenedor flex.

flex-justify-flex-start
flex-justify-center
flex-justify-flex-end
            <div class="flex flex-row flex-justify-flex-start">
                <div class="badge success">flex-justify-flex-start</div>
            </div>
    
            <div class="flex flex-row flex-justify-center">
                <div class="badge success">flex-justify-center</div>
            </div>
    
            <div class="flex flex-row flex-justify-flex-end">
                <div class="badge success">flex-justify-flex-end</div>
            </div>


Ejemplo con 'space-between'

En este ejemplo, los elementos se distribuyen equitativamente a lo largo del contenedor, con espacios entre ellos.

Elemento 1
Elemento 2
Elemento 3
            <div class="flex flex-row flex-justify-space-between">
                <div class="badge success">Elemento 1</div>
                <div class="badge success">Elemento 2</div>
                <div class="badge success">Elemento 3</div>
            </div>


Ejemplo con 'space-around'

En este ejemplo, los elementos se distribuyen equitativamente a lo largo del contenedor, con espacios alrededor de ellos.

Elemento 1
Elemento 2
Elemento 3
            <div class="flex flex-row flex-justify-space-around">
                <div class="badge success">Elemento 1</div>
                <div class="badge success">Elemento 2</div>
                <div class="badge success">Elemento 3</div>
