Contenedores

Los container estan divididos en 3 tipos: container, container-(s,m, l, xl y xxl) y container-100

Container

Este container tiene la capacidad de ajustarse en cada uno de los "stoppoint", que son simplemente las resoluciones predefinidas, que son:

s: Tamaño <= 540px

m: Tamaño <= 720px

l: Tamaño <= 960px

xl: Tamaño <= 1200px

xxl: Tamaño <= 1400px

Cuando el container alcanza uno de estos puntos de alto, se ajusta a 250px menos que su tamaño actual para evitar desbordamientos. Sin embargo, cuando llega a 540px, su tamaño se establece en el 100% del espacio disponible en el container padre. Esta configuración se declara en la clase como "container".

Container-(s, m, l, xl y xxl)

La funcionalidad de estos containers es la siguiente: cuando su tamaño excede su resolución mínima, se ajustan a 250px menos que su punto de detención (stoppoint), pero al alcanzar este punto (stoppoint), ocupan el 100% del espacio disponible en la pantalla.

Cuando un contenedor alcanza uno de estos puntos de parada (stoppoint), se ajusta a una longitud 250px menor que su tamaño actual para evitar desbordamientos. Sin embargo, cuando alcanza una anchura de 540px, su tamaño se establece en el 100% del espacio disponible en el contenedor padre. Esta funcionalidad se declara en la clase con un subfijo específico, como "container-s" o "container-m".

Container-100

Este container tiene la capacidad de alcanzar el 100% del contenedor padre, eliminando cualquier tipo de margin o padding, y ocupando todo el espacio disponible. Es útil para ser utilizado como un containerr principal o un div donde se requiera ocupar el 100% del tamaño disponible.

Filas y columnas

El empleo de filas y columnas contribuye significativamente a la capacidad de respuesta del diseño de una página web. En esta biblioteca, ofrecemos un sistema de modelado de columnas y filas. Para empezar, es necesario crear un container principal con la clase "fila". Por ejemplo, utilizando un elemento div como este: <div class="fila"></div> dentro de este container, se define también el modelo de columnas, como se muestra a continuación: <div class="fila columna-5"></div> cada elemento dentro de este div se ajustará al modelo de 5 columnas, y las filas se generan automáticamente. Los tipos de columnas disponibles son los siguientes:

columna-5: Se generan 5 columnas

columna-4: Se generan 4 columnas

columna-3: Se generan 3 columnas

columna-2: Se generan 2 columnas

columna-1: Se genera una sola columna

Cuando las columnas alcanzan los "puntos de alto" mencionados anteriormente, se adaptan una por una. Por ejemplo, al llegar al primer punto de alto, una columna de tamaño 5 se reducirá a 4, y así sucesivamente, hasta que solo quede 1 columna.
