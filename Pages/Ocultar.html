Ocultar elementos

Para ocultar o mostrar los elementos, simplemente se agrega la clase correspondiente ocultar y para mostrar los elementos ocultos solo se quita la clase correspondiente mostrar ejemplo:

Este elemento esta oculto

En el ejemplo previo, el div no es visible ya que está oculto, sin embargo, continúa ocupando espacio en la página. Puedes verificar esto inspeccionando la página con Ctrl + Shift + I o F12 y observando su presencia en el código fuente..

Codigo

<< /span>div class="titulo-principal shadow-3 mt-3 border ocultar">Este elemento esta oculto << /span>/div>

Mostrar elementos ocultos

Para mostrar elementos que están ocultos, simplemente se agrega la clase correspondiente mostrar si aplicamos la clase adecuada, los elementos ocultos se mostrarán en pantalla. Ahora, para mostrar el div del ejemplo anterior, simplemente eliminamos la clase que lo estaba ocultando. Si la clase de oculto es "ocultar", entonces basta con quitar esa clase del elemento div.

Este elemento esta oculto

Como puedes ver, el objeto, en este caso el div, vuelve a aparecer en la pantalla tal como habíamos previsto, una vez que hemos quitado la clase de ocultar.

Codigo

<< /span>div class="titulo-principal shadow-3 mt-3 border mostrar">Este div esta oculto. << /span>/div>

Esta práctica resulta especialmente útil para gestionar la visibilidad de elementos en diferentes contextos o en respuesta a eventos específicos. Al agregar la clase de ocultar, los elementos se vuelven invisibles para el usuario, mientras que al quitarla, vuelven a ser visibles. Esto proporciona un método sencillo y efectivo para controlar la presentación de contenido en una página web.
